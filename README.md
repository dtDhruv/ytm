# ytm

Stream YouTube audio from your terminal.

## Requirements

- [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- [mpv](https://mpv.io/)

## Install

```
uv tool install .
```

## Usage

```
ytm                    # Interactive mode
ytm search <query>     # Search and pick
ytm play <url>         # Play a URL directly
```
